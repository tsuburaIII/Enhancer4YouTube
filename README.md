# 🎬 Enhancer4YouTube

YouTube を快適に閲覧するためのカスタムスクリプトです.  
指定したキーワードを含む動画を自動的に非表示にします.

---

## ✨ 機能概要

### 🚫 動画フィルタリング
- タイトル / チャンネル名に**指定したNGワード**を含む動画を自動で非表示にします.
- ホーム / 検索結果 / サイドバー などに対応.
- スクロールやページ遷移など、動的に読み込まれる動画も監視します.
- デバッグモードでは非表示理由を可視化できます.

---

## 🔧 動作環境

- PC版 YouTube
- [User JavaScript and CSS](https://chromewebstore.google.com/detail/user-javascript-and-css/nbhcbdghjpllgmfilhnhkllmkecfmpld)（拡張機能）
- ※ 主に **Chrome 環境での利用を想定**しています.

---

## 📥 導入方法

1. **User JavaScript and CSS** 拡張機能をインストールします.
2. 拡張機能の管理画面を開き、新規スクリプトを作成します.
3. `Enhancer4YouTube.js` の内容をすべて貼り付けます.
4. 対象URLを以下のように設定します.

```

https://www.youtube.com/*

```

5. 保存後、YouTube を再読み込みしてください.

---

## ⚙️ 設定・カスタマイズ

### NGワードの編集

```js
const NG_WORDS = [
  "キーワード",
  "チャンネル名",
  "タイトルの一部"
];
````

* 大文字・小文字は区別されません.
* 部分一致で判定されます.

### デバッグモード

```js
const DEBUG_MODE = true;
```

* `true`：赤枠とラベルで非表示理由を表示します.
* `false`：対象要素を完全に非表示にします.（通常はこちら）
